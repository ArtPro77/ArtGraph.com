Программа для создания рисунков на графике

Представляю вашему вниманию программу для создания рисунков на графике...

  1. Создание точек - ЛКМ

  2. Удаление точек - ПКМ

  3. Соединение двух точек - C/с

  4. Сохранение/Загрузка графиков

  5. Уникальный формат файла: .gr

  6. Поддержка трёх языков: 1. Русский(по умолчанию) 2.Английский 3. Украинский

  7. Синтаксис .gr файла при открытии через txt:
    ‹GraphFile›
    name: имя_файла
    ‹P›
    points: количество точек
    (x,y)
    (x,y)
    (x,y)
    ‹S›
    lines: количество линий
    (x,y) -> (x,y)
    (x,y) -> (x,y)
    (x,y) -> (x,y)

  8. Ввод координат через ENTER

  9. Отображение координат отключить/включить N/т

Рекомендуется: работать на компьютерах(рекомендуются экраны: 3440х1440 1920х1080)
